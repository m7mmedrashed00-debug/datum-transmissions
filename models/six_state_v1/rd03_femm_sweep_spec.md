# RD-03 FEMM Sweep Specification v1.0 — conservative electromagnetic map for v0.3a

**Status: SPECIFICATION ONLY. No FEMM run is authorized by this document.**
Written before any solve, per the pathway's gate discipline. Implements Sol 5.6
correction #3: ONE scalar co-energy surface; λ and F are derived, never
independently interpolated. References: [Meeker-FEMM] (manual in hand),
[pyFEMM]/OctaveFEMM (MATLAB-driven), [Kallenbach], [SMAG] (B–H data).

## 1. Objective
Produce a versioned, mesh-converged map W′(i, x, T) (magnetic co-energy) for the
baseline proportional solenoid geometry such that:

    λ(i,x,T) = ∂W′/∂i        (flux linkage)
    F(i,x,T) = ∂W′/∂x |i,T   (plunger force)

satisfy reciprocity (∂λ/∂x = ∂F/∂i, both = ∂²W′/∂i∂x) by construction, closing
model defect D1 when substituted into the v0.3a plant:
V = R(T)·i + dλ/dt,  m·dv/dt = F − ...

Quasi-static and CONSERVATIVE by design: hysteresis and eddy-current losses are
EXCLUDED from the map (they need history states; single-valued maps must not
imply them).

## 2. Geometry parameterization (all [Repr] until a donor solenoid is measured)
Axisymmetric (r,z) model. Parameters, each with source recorded in the registry:
plunger radius r_p; plunger length l_p; radial working gap g_r; axial stroke
coordinate x ∈ [0, g_max]; pole/flux-washer dimensions; coil window (r_in, r_out,
height, N turns, fill factor); core/housing steel path sections; non-magnetic
spacer thickness. Geometry file is generated by ONE parameterized builder script
(OctaveFEMM `mi_*` calls) — no hand-drawn geometry, ever.

## 3. Materials
- Core/plunger/housing: low-carbon steel B–H curve from [SMAG] (named grade
  recorded; processing-state caveat per [Kasai2023] noted in the registry row).
- Coil: copper, σ(T) via [Dellinger] alpha — used OUTSIDE FEMM for R(T); the
  magnetostatic solve uses imposed coil current, not voltage.
- Air/oil gap: μr = 1 (ATF is non-magnetic).
- Temperature handling: FEMM solves are ISOTHERMAL snapshots. T enters the map
  ONLY through material-property selection per T grid point (B–H temperature
  derating if source data exist; else T-dependence deferred to measurement and
  the map is W′(i,x) with a declared T-validity note — do NOT fabricate B–H(T)).

## 4. Boundary conditions & solver settings
- Open boundary via Kelvin transformation ring (per [Meeker-FEMM] manual §
  open-boundary guidance) OR asymptotic boundary condition; choice recorded.
- Axisymmetric magnetostatic problem; precision 1e-8; smart-mesh off (explicit
  mesh sizing for reproducibility).

## 5. Sweep grid
- Current: i = 0 : 0.1 : 2.0 A (21 points; refine near saturation knee after
  first pass).
- Stroke: x = 0 : 0.2 : 4.0 mm (21 points; refine near x = 0 where ∂F/∂x is
  largest).
- Temperature: {25, 80, 120} °C IF material data support it (else single-T map,
  declared).
- Total first-pass solves: 21 × 21 (× nT). Batch driver: OctaveFEMM from MATLAB
  (MATLAB MCP orchestrates; CI consumes the VERSIONED map artifact and never
  runs FEMM per test).

## 6. Outputs per solve (recorded, not post-hoc)
λ from FEMM circuit-property flux linkage of the coil; F via BOTH (a) weighted
stress tensor on the plunger and (b) later ∂W′/∂x differentiation of the
assembled co-energy surface. (a) vs (b) agreement is an acceptance criterion.

## 7. Co-energy assembly (the single-surface rule)
1. At each (x,T): W′(i,x,T) = ∫₀^i λ(ι,x,T) dι by cumulative trapezoid over the
   i-grid (monotone λ(i) required — acceptance check).
2. Fit/spline W′ over (i,x) with a shape-preserving scheme (PCHIP first; smoothing
   spline only with recorded justification).
3. Derive λ = ∂W′/∂i and F = ∂W′/∂x from the SAME surface object.
4. Export: `rd03_emmap_vX.mat` containing W′ grid + derivation metadata + mesh
   record + SHA-256; registry row mandatory.

## 8. Acceptance criteria (gate v0.3a exit evidence)
- Mesh convergence: force and λ change < 0.5 % on one uniform refinement at the
  worst-case corner (max i, min gap); record the convergence pair.
- Reciprocity residual: max |∂λ/∂x − ∂F/∂i| / |∂F/∂i| < 1 % over the interior grid.
- Force-method agreement: weighted-stress-tensor F vs ∂W′/∂x within 2 % interior
  (edges excluded, recorded).
- Physical sanity: λ(i) monotone at every (x,T); F ≥ 0 toward closing; saturation
  knee location stable under refinement.
- Energy ledger: with the map installed in v0.3a, testXFAIL_D1 FAILS (defect
  fixed) and is promoted to a positive conservation test with residual < declared
  tolerance.
- Every artifact (geometry script, mesh record, raw solves, assembled map) has a
  Model Evidence Registry row with SHA-256.

## 9. Explicitly out of scope
Hysteresis, eddy currents, 3-D leakage paths, thermal solve coupling, COMSOL.
Trigger to revisit: measured dynamic λ(i) loops (E1 family) showing loss widths
inconsistent with the conservative model beyond the declared uncertainty.
